# HashGen Pro Ultimate

A powerful and modern hash generator web application built with React and Vite.

## Features

- **Multiple Hash Algorithms**: MD5, SHA-1, SHA-256, SHA-512, SHA-3, RIPEMD-160
- **Text Input**: Generate hashes from any text input
- **File Upload**: Hash entire files instantly
- **Copy to Clipboard**: One-click copy for all generated hashes
- **Beautiful UI**: Modern gradient design with Tailwind CSS
- **Real-time Generation**: Hashes update as you type

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Technologies Used

- React 18
- Vite
- Tailwind CSS
- CryptoJS
- Lucide React Icons
